## 🐦‍⬛ Crow Innovation Technologies
**El-Piojo Web** es una plataforma sin fines de lucro diseñada para optimizar la compra y venta de ropa de segunda mano. Buscamos fortalecer la economía circular de los universitarios, ofreciendo una solución ética y tecnológica para la comunidad local y personas foráneas.
